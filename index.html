¿Cómo jugar 2048?

El objetivo es conseguir una ficha con el número 2048 combinando números iguales:

  • 🔢 Los números iguales se suman al juntarse
  • 2️⃣ Empieza con fichas de 2 y 4
  • ➕ Al mover, los números iguales se combinan
  • 🎯 Cada movimiento genera una nueva ficha
  • 🏆 ¡Consigue el 2048 para ganar!

Instrucciones para móvil

Desliza el dedo sobre la pantalla en cualquier dirección:

  • 👆 Desliza hacia arriba para mover arriba
  • 👇 Desliza hacia abajo para mover abajo
  • 👈 Desliza hacia la izquierda para mover izquierda
  • 👉 Desliza hacia la derecha para mover derecha
